# @rir360/ui

A collection of **clean**, **flexible**, and **reusable** UI components optimized for Next.js.

Built with **TypeScript** and **Tailwind CSS**, designed to be the perfect foundation for any modern web project.

---

## 🚀 Quick Start

Add components to your project instantly using the CLI:

```bash
npx @rir360/ui add <component-name>
```

For example, to add the Splide carousel:
```bash
npx @rir360/ui add splide
```

---

## 🧱 Available Components

| Component | Description |
| :--- | :--- |
| `header` | Sticky navigation with logo, mobile drawer, and optional banner. |
| `hero` | Impactful hero section with title, description, and media. |
| `section` | Base layout wrapper for consistent spacing and width. |
| `editorial` | Versatile two-column layout for text and images (reversible). |
| `grid` | Responsive grid system for content organization. |
| `card` | Versatile card component with image, title, and CTA. |
| `contact-form` | Ready-to-use form with basic validation and Tailwind styling. |
| `splide` | Optimized Splide carousel wrapper with auto-scroll support. |
| `footer` | Clean footer with navigation links, social icons, and copyright. |

---

## ✨ Features

- **Base-First Design**: Components act as solid foundations, not restrictive templates.
- **Fully Responsive**: Mobile-first architecture out of the box.
- **Type Safe**: First-class TypeScript support for a better developer experience.
- **Tailwind Integrated**: Easily customize everything using standard Tailwind classes.
- **Next.js Optimized**: Designed with App Router and SSR in mind.

| Command | Usage |
